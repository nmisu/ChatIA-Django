Ayuda y Documentación

Bienvenido a la página de información de ChatIA. Aquí encontrarás todos los detalles sobre el funcionamiento y la arquitectura del proyecto.

Funcionamiento de la aplicación

ChatIA es un asistente conversacional diseñado para ofrecer respuestas rápidas y mantener el contexto de tus charlas. Para empezar a usarlo:

  1. Inicia sesión o Regístrate: Necesitas una cuenta para que tu historial se guarde de forma privada.
  2. Crea un nuevo chat: Usa el botón de la página principal para abrir una sala.
  3. Elige tu modelo: En la ventana de chat, selecciona qué IA quieres usar (desde modelos ultrarrápidos hasta razonadores complejos).
  4. Gestiona tus datos: Puedes renombrar conversaciones, eliminarlas o descargar el historial completo en formato JSON usando el botón superior del chat.

Estructura Técnica

Este proyecto se ha construido siguiendo el patrón Modelo-Vista-Plantilla (MVT) y utiliza las siguientes tecnologías:

  • Backend: Desarrollado en Python 3 utilizando el framework Django.
  • Base de Datos: Almacenamiento relacional persistente mediante SQLite3.
  • Frontend y Reactividad: Maquetación responsive con Bootstrap 5 y actualizaciones asíncronas de la interfaz (sin recargar la página) gracias a HTMX.
  • Inteligencia Artificial: Integración mediante API REST con los servicios de inferencia de NVIDIA NIM, procesando payloads en formato JSON.
